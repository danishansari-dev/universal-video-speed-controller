How It Works in 4 Quick Steps

Configure the extension and speed up videos across the web in less than a minute.

01

Install

Add the extension to Chrome by clicking the install CTA from the Chrome Web Store.

02

Pin Extension

Click the extensions puzzle icon in your toolbar, then click the pin icon next to our logo.

03

Play a Video

Open any HTML5 video player (like YouTube). Hover to reveal the speed widget or use hotkeys.

04

Customize

Click the pinned icon to customize shortcuts, domains list rules, or toggle mouse gestures.

Controls Reference

Default Key Bindings & Actions

Configure these defaults from the extension dashboard popup at any time.

Shortcut / Command Triggered Action Details / Behaviors
] or Shift + . Speed Up Increases speed rate by 0.25x steps (up to 10x).
[ or Shift + , Slow Down Decreases speed rate by 0.25x steps (down to 0.25x).
\ Reset Speed Immediately restores playback speed rate back to normal 1x.
Hold X Temporary Boost Mode Speeds up video to 2x (customizable) while pressed, returns on release.
Shift + S Toggle Widget Hides or reveals the floating controls widget overlay on video.
Shift + H Toggle Overlay HUD Toggles the fullscreen visual overlay speed toast notifications.
Ctrl + Mouse Scroll wheel Wheel Speed Control Scroll over player boundaries while holding Ctrl to adjust rate.
Pinch In / Pinch Out Touchpad Gestures Pinch gestures on supported touchpads adjust speed rates dynamically.

Developer Onboarding: Unpacked Installation

If you want to build custom features or run the extension locally from the source files, follow these steps:

  1. Download or clone the source files from our GitHub Repository.
  2. Launch Chrome and navigate to the extension panel: chrome://extensions.
  3. Enable Developer Mode using the toggle switch located in the upper right-hand corner.
  4. Click the Load Unpacked button and select the root directory containing the project folder.
  5. Open any page with an HTML5 video player to run and test the unpacked extension.
